# AZI Probability Calculator

Аналитический инструмент на Python для расчёта **точной вероятности победы**
в карточной игре **Ази** при неполной информации о картах оппонента.

Проект моделирует конечное дерево игровых состояний и использует
полный перебор допустимых комбинаций карт противника.

---

## Описание задачи

Дано:
- карты игрока на руках (3 карты),
- известная козырная карта,
- неизвестные карты оппонента.

Требуется:
- вычислить вероятность победы игрока
- при фиксированном первом ходе
- при условии, что оппонент может иметь любую допустимую комбинацию карт.

---

## Подход и метод

### Комбинаторный анализ
Перебираются все возможные наборы карт оппонента,
совместимые с оставшейся колодой (27 карт).

### Дерево исходов
Для каждого набора карт строится полное дерево возможных ходов
на 3 раунда с учётом:
- обязательного хода в масть,
- использования козыря,
- правил определения победителя взятки.

### Стратегии первого хода
Рассматриваются различные стратегии:
- сильнейшая карта,
- слабейшая карта,
- средняя карта.

### Детерминированная логика
Правила игры и сравнения карт реализованы без случайности,
что позволяет получить **точный**, а не приближённый результат.

Пример использования
from azi import overall_probability_towin

my_cards = [('9', 'S'), ('Q', 'D'), ('6', 'H')]
trump = ('A', 'S')

probability = overall_probability_towin(
    my_cards=my_cards,
    trump=trump,
    first_move=None,
    my_first_turn=False
)

print(f"Вероятность победы: {probability}%")
